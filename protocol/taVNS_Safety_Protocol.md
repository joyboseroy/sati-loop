# taVNS Safety Protocol (Layer 3, Research Use Only)

## Contraindications

Exclude participants with: cardiac pacemaker, implanted neural device, pregnancy, active epilepsy, skin lesion at electrode site.

## Session parameters

| Parameter | Value |
|-----------|-------|
| Current | 0.5–2.0 mA (manual, not software-adjustable) |
| Frequency | 25 Hz |
| Pulse width | 250 microseconds |
| TDM | 400ms on / 100ms off |
| Trigger | Gross agitation >60 continuous seconds |
| Duration per trigger | 180 seconds maximum |
| Session maximum | 30 minutes active stimulation |

## Adverse effect monitoring

After each session, collect self-report: electrode site sensation, headache, mood changes, sleep quality. Two consecutive moderate/severe reports: discontinue taVNS and consult clinician.
