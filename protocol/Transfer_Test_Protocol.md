# Transfer Test Protocol

## Purpose

The only success criterion for Sati-Loop. Determines whether Delta t (distraction to self-correction interval) persists in device-absent practice.

If Delta t improves only with the device active: dependency, not skill. This is a failure condition.

## Phases

**Phase A (Weeks 1-2):** Passive sensing, no cues. Phenomenological probes at random intervals. Baseline Delta t established.

**Phase B (Weeks 3-8):** Active closed-loop with scaffold withdrawal. One device-absent session per week measures transfer trajectory.

**Phase C (Weeks 9-10):** Device physically absent. No sensing. Delta t measured via self-timed phenomenological prompts only.

**Phase D (Month 3, Month 6):** Optional long-term follow-up.

## Success criterion

    Delta t (Phase C) < Delta t (Phase A)

Statistical test: paired t-test or Wilcoxon signed-rank, alpha = 0.05.

## Control condition

Sham device delivers cues at random intervals uncorrelated with detected distraction. Controls for expectation and interruption effects.
