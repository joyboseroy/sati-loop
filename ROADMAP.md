# Roadmap

## Milestone 1 — Signal validation
- BrainFlow streaming from Muse S and OpenBCI
- Basic preprocessing and artifact rejection
- Validate EEG quality from dry electrodes
- Stream PPG, respiration, IMU alongside EEG

## Milestone 2 — Calibration and classifier
- Phase A passive sensing with phenomenological probes
- Label collection from self-reports
- Train personalised Random Forest classifier
- Validate classifier on held-out calibration sessions

## Milestone 3 — Minimal cue loop
- Subtractive acoustic cue delivery via bone conduction
- Cue decision logic with confirmation window
- Basic scaffold withdrawal (threshold widening)
- Log all events for transfer analysis

## Milestone 4 — Transfer testing
- Weekly device-absent session protocol
- Compare Delta t in assisted vs unassisted sessions
- Statistical analysis of transfer vs dependency
- First prototype paper if results are positive

## Milestone 5 — Layer 2 integration
- HRV-guided breath pacing
- Resonant frequency calibration protocol
- Passive ambient audio (non-contingent)
- Validate that Layer 2 does not create implicit reward

## Milestone 6 — Open dataset release
- Anonymised Phase A calibration recordings
- Labelled phenomenological self-reports
- Transfer test longitudinal data
- Public release under CC BY 4.0

## Milestone 7 — taVNS research module (research only)
- TDM gating firmware validation
- Isolated stimulator PCB design
- IRB protocol for supervised testing
- Pilot study under controlled conditions
