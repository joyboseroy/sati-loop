# Sati-Loop Philosophy

Sati-Loop is a closed-loop attentional scaffolding system deliberately designed to make itself unnecessary over time.

## What it is not

- Not a meditation app
- Not a relaxation device
- Not a brain optimisation tool
- Not an enlightenment machine
- Not a consumer wellness product

## The foundational problem

From the companion paper *The Contemplative Alignment Problem* (Bose, 2026):

> When a measurable proxy becomes the target of a feedback loop, the practitioner's brain will optimise for the proxy, not the intended capacity.

Calm EEG can be produced by genuine settled awareness, mild dissociation, pleasant dullness, or a specific muscle configuration that creates the target waveform as an artefact. A device that rewards all of these equally is training proxy production, not meditation.

## The engineering response

**Negative-only feedback.** The only signal the device delivers is a cue when distraction is detected. There is no reward for calm.

**No proxy targets.** The device does not try to increase alpha power, HRV coherence, or any other biomarker as an end in itself.

**Scaffold withdrawal.** The intervention systematically fades as skill develops.

**Transfer as the only metric.** Whether the skill persists without the device is the only question that matters.

## The core constraint

> The intervention engine must never optimise directly against the classifier output.

This constraint generalises beyond meditation devices. It applies wherever a measurable proxy substitutes for an unmeasurable target in an adaptive system with a human in the loop.
