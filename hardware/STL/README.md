# 3D Printable Parts

STL files for the headband enclosure and electrode mounts will be added here.

## Planned parts

- Adjustable headband frame (PLA or PETG)
- Frontal electrode mount with spring contact
- Rear electronics enclosure
- Bone conduction transducer mount (temporal position)
- Cable clip and loom guide

## Status

Not yet available. Design in progress.

## Print settings (planned)

- Material: PLA for initial prototype; PETG for durability
- Layer height: 0.2mm
- Infill: 20% for enclosure; 40% for structural parts
- Support: minimal; designs will aim to be support-free where possible
