# Datasets

This directory contains placeholder structures and documentation for future experimental datasets collected using the Sati-Loop platform.

No participant data is currently included in this repository.

## Planned Dataset Modalities

- EEG
- PPG / HRV
- Respiration
- IMU / posture
- Phenomenological probe labels
- Cue event timestamps
- Transfer test outcomes

## Planned Use Cases

- Mind-wandering onset estimation
- Artifact rejection benchmarking
- Transfer-learning validation
- False-positive analysis
- Cross-subject variability analysis

## Privacy and Ethics

Future datasets must:
- remove personally identifying information,
- avoid raw audio/video recording,
- use randomized participant IDs,
- follow informed consent procedures.

See:
- `protocol/`
- `safety/`
- `docs/KNOWN_LIMITATIONS.md`
